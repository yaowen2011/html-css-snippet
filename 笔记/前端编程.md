## 格式化时间的函数
```javascript
function formatTime(time) {
  var [ hours, minute, seconds ] = [
    time.getHours(),
    time.getMinutes(),
    time.getSeconds()
  ].map(num => num < 10 ? '0' + num : num);

  return {hours, minute, seconds};
}
```
