visibility:hidden; disdplay:none; 区别 In the latter case, the ele- ment is not displayed and is also removed from the document altogether so that it doesn’t have any effect on document layout. 两者都是不可见， 前者依然在文档流中占着位置； 后者则被从文档流中移除，不再影响文档流的布局。
display: 不再保留文档中的位置 visibility: 依然保留文档流中的位置 visible|hidden overflow: visible|auto|scroll |hidden scroll ：右、下都显示滚动条
