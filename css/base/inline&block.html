css引入方式

1、行内式:style="" 2、内嵌式:一定程度上，实现结构样式分离 3、外链式

block元素 inline元素 inline-block元素 三种模式转换

block（块）元素 比如：div、p 、h1 属性特点： 0、独占一整行 1、宽度默认填充满containing block（父容器） 2、不设高，高度由内容决定 3、可以设置 width 、height 、上下 左右的内外边距 4、可以容纳任何其他标签

inline(行内) 比如： span、a、strong 属性特点： 1、不能设置 width、height属性 2、一行可以放多个 3、行内不能放块级别元素 4、width 、height 内容决定 5、可以设置水平方向的padding 、margin 特例： 1、a 可以放img span div 等标签 注意：a链接内不能再放a链接，违背设计原理 2、p h1-h6只能放文字块，不能放其他的块级元素 如：p标签内不能放div标签

inline-block(行内块元素) 比如：img 、input 、td 简单理解：放在行内的块 1、可以设置 width 、height 2、一行可以放置多个 3、同行相邻 会产生缝隙

div { display:inline; } 块级元素 转成行内元素 width height 失效 span { display:block; } 行内转成块级元素 span { display:inline-block; } 行内元素 可以设置垂直方向的padding ，但是垂直方向的padding不占空间 不占位置 背景色是有效的 高度发生变化
