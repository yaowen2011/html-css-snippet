目标伪类-selector

复合选择器 1.交集选择器(两个同时满足) h3.class span#sub-id .class1.class2 .class1#idname 注意： 有标签选择器，标签必须放在前面（污染变量名） 2、并集选择器 h3, div, span {....} 3、后代选择器 (descendants selector) ul li 4、子元素选择器 ul>li ul>li>span>a

属性选择器(css3) 1、E[attr] :存在attr属性即可选取； 2、E[attr="radio"] : 属性值为radio的即可选取； 3、E[attr*='ve']: 只要找到ve 字符就匹配； 4、E[attr^='radio']: radio 开始就能匹配到； 5、E[attr$='radio']: 以radio结尾的 就能找到； 6、 attr~="radio" type~="hot" 使用空格分割的多属性 type="hot hot2" 必须与分割出来的属性完全相等， 注意：完全相等 7、li[price|='very'] 类似于very very- 就可以别选中 可以选中 very very-content1 very-content2 这个very 还必须是写在最左边的属性 完全的css国际命名规范 才有作用
